AngularJS [:syntax] Syntax Error: Token ':' is an unexpected token

[$parse:syntax] Syntax Error: Token ‘:’ is an unexpected token at column 11 of the expression

坑爹的异常报错,

解决办法:
step1: 去掉jsp中的取参数的 el 语句 ,${}, 可以改为在angular的 $scope中进行传递
step2: 替换ng-app定义范围内的标签元素的原生属性为angularJS中固有的, 如onclick 改为 ng-click, src 改为 ng-src

问题解决! bingooooooooooooooo!
