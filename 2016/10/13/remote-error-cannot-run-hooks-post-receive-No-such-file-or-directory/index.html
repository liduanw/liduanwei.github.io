remote: error: cannot run hooks/post-receive: No such file or directory

remote: error: cannot run hooks/post-receive: No such file or directory

解决办法:http://stackoverflow.com/questions/3563904/git-hook-post-merge-error-cannot-run

大体意思就是说sh文件中存在不可见的非unix空白控制符(换行符,空格等), 所以去掉即可,
(可以使用 cat -v /path/file 查看所有空白控制符,如windows中的换行会显示成^M)
